Готовий до використання код **Музею v4 (SPA)** в одному файлі.
У цій версії я поєднав:
 1. **JSON-двигун**: Весь контент керується масивом даних у скрипті.
 2. **Адаптивне меню**: Повноцінне «бургер-меню» для смартфонів.
 3. **SPA-логіку**: Миттєве перемикання між залами без перезавантаження.
 4. **Модальні вікна**: Детальний перегляд кожного експоната.
### Як користуватися цим файлом:
 1. **Наповнення**: У скрипті в розділі MUSEUM_DATA.exhibits просто додавайте нові об'єкти (id, hall, title, short, full, img).
 2. **Зображення**: В полі img вказуйте шлях до ваших фото (наприклад, img: "img/exponat1.jpg").
 3. **Адаптивність**: На телефонах з'явиться кнопка «☰», яка відкриває список залів.
 4. **SPA**: Користувач переходить між «Стінами» без перезавантаження сторінки, що створює відчуття сучасного застосунку.
